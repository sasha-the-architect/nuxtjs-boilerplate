## Setup

Make sure to install the dependencies:

```bash
# pnpm (recommended)
pnpm install

# npm
npm install

# yarn
yarn install
```

## Development

Start the development server on http://localhost:3000

```bash
pnpm run dev
```

## Code Quality

This boilerplate includes several code quality tools:

```bash
# Run linting
pnpm run lint

# Fix linting issues
pnpm run lint:fix

# Format code
pnpm run format
```

## Production

Build the application for production:

```bash
pnpm run build
```

Locally preview production build:

```bash
pnpm run preview
```

## Deployment

This boilerplate is ready for deployment on any platform that supports Node.js:

- [Vercel](https://vercel.com)
- [Netlify](https://netlify.com)
- [Railway](https://railway.app)
- [Digital Ocean App Platform](https://www.digitalocean.com/products/app-platform)

Checkout the [Nuxt deployment documentation](https://nuxt.com/docs/getting-started/deployment) for more information.
